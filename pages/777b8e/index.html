# 重新k8s初始化

1、移除所有工作节点

kubectl delete node [name]

2、所有工作节点删除工作目录，并重置kubeadm

rm -rf /etc/kubernetes/*
kubeadm reset

3、Master节点删除工作目录，并重置kubeadm

rm -rf /etc/kubernetes/*
rm -rf ~/.kube/*
rm -rf /var/lib/etcd/*

rm -rf /var/lib/cni/

rm -fr /etc/cni/net.d

kubeadm reset -f

4、重新init kubernetes

# DNS子域名

很多资源类型需要可以用作 DNS 子域名的名称。 DNS 子域名的定义可参见 RFC 1123 (opens new window)。 这一要求意味着名称必须满足如下规则：

  • 不能超过253个字符
  • 只能包含小写字母、数字，以及'-' 和 '.'
  • 须以字母数字开头
  • 须以字母数字结尾
